✅ "Direct Access Not Permitted" Error - FIXED!

Problem: helpers-extended.php was checking for ROOT_PATH constant that wasn't defined.

Solution: Changed the security check to look for SITE_NAME constant instead (which is defined in config.php).

Result: All pages should now load without the "Direct access not permitted" error!

If you see database connection errors, make sure:
